# SBAN v20 Demo

This folder is the newcomer-facing continuing-session demo surface for SBAN v20.

## What it does

- packages the current `chat-demo` binary flow for new users
- preserves one chat session across turns through `session_path`
- bundles the v20 dialogue seed, prompt list, and scripted session evaluation file
- keeps the release caveat explicit: the packaged numeric benchmark is self-seeded and transductive

## Local source build demo

From the repo root after `zig build -Doptimize=ReleaseSafe`:

```bash
zig-out/bin/zig_sban chat-demo "hi im tom" 160 mode=free seed_path=data/sban_dialogue_seed_v20.txt session_path=session_v20.txt
zig-out/bin/zig_sban chat-demo "can you recall my name" 160 mode=free seed_path=data/sban_dialogue_seed_v20.txt session_path=session_v20.txt
```

## Packaged newcomer demo

The release bundles generated by `scripts/package_v20_demo.py` rename the binary to `sban_v20` or `sban_v20.exe` and include:

- `SBAN_v20_Start.bat` for Windows
- `SBAN_v20_Start.sh` for Linux
- `README_FIRST.txt`
- `sample_prompts_v20.txt`
- the bundled v20 seed, prompt, and session-eval assets

## Important caveat

The bundled chat demo is a user product surface.

The packaged numeric release documented elsewhere in the repository is a separate benchmark protocol and is explicitly self-seeded and transductive. Keep that distinction clear when discussing the v20 scores.
