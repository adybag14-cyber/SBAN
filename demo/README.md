# SBAN v21 Demo

This folder is the newcomer-facing grounded continuing-session demo surface for SBAN v21.

## What it does

- packages the current `chat-demo` binary flow for new users
- preserves one chat session across turns through `session_path`
- bundles the v21 dialogue seed, prompt list, and scripted session evaluation file
- enables automatic CPU or GPU retrieval selection through the runtime `backend=auto` path
- keeps the benchmark caveat separate from the user-facing chat story

## Local source build demo

From the repo root after `zig build -Doptimize=ReleaseSafe`:

```bash
zig-out/bin/zig_sban chat-demo "hi i am tom and i need help" 160 seed_path=data/sban_dialogue_seed_v21.txt session_path=session_v21.txt backend=auto
zig-out/bin/zig_sban chat-demo "can you recall my name" 160 seed_path=data/sban_dialogue_seed_v21.txt session_path=session_v21.txt backend=auto
zig-out/bin/zig_sban chat-demo "my favorite color is blue" 160 seed_path=data/sban_dialogue_seed_v21.txt session_path=session_v21.txt backend=auto
zig-out/bin/zig_sban chat-demo "what is my favorite color" 160 seed_path=data/sban_dialogue_seed_v21.txt session_path=session_v21.txt backend=auto
```

## Packaged newcomer demo

The release bundles generated by `scripts/package_v21_demo.py` rename the binary to `sban_v21` or `sban_v21.exe` and include:

- `SBAN_v21_Start.bat` for Windows
- `SBAN_v21_Start.sh` for Linux
- `README_FIRST.txt`
- `sample_prompts_v21.txt`
- the bundled v21 seed, prompt, and session-eval assets
- captured v21 recall, uncertainty, session-eval, and acceleration examples when they are available

## Important caveat

The bundled chat demo is a user product surface.

The packaged numeric release documented elsewhere in the repository is a separate benchmark protocol. Keep that distinction explicit when discussing the v21 scores.
