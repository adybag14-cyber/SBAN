# SBAN v19 Demo

This folder is the newcomer-facing demo surface for SBAN v19.

## What it does

- packages the current `chat-demo` binary flow for new users
- ships a starter script for Windows and Linux
- bundles the v19 dialogue seed and prompt list
- keeps the release caveat explicit: the packaged numeric benchmark is self-seeded and transductive

## Local source build demo

From the repo root after `zig build -Doptimize=ReleaseSafe`:

```bash
zig-out/bin/zig_sban chat-demo "what is SBAN v19" 160 seed_path=data/sban_dialogue_seed_v19.txt
```

## Packaged newcomer demo

The release bundles generated by `scripts/package_v19_demo.py` rename the binary to `sban_v19` or `sban_v19.exe` and include:

- `SBAN_v19_Start.bat` for Windows
- `SBAN_v19_Start.sh` for Linux
- `README_FIRST.txt`
- `sample_prompts_v19.txt`
- the bundled v19 seed and prompt assets

## Important caveat

The bundled chat demo is a user product surface.

The packaged numeric release documented elsewhere in the repository is a separate benchmark protocol and is explicitly self-seeded and transductive. Keep that distinction clear when discussing the v19 scores.
